P: "Deu erro de autenticação"
R: Use Personal Access Token (PAT) no GitHub: Settings → Developer settings → 
   Tokens → New token (classic) → Marque 'repo'
   Depois use: 
   git remote set-url origin https://TOKEN@github.com/USUARIO/fraga-dashboard.git

P: "Como ver se o push funcionou?"
R: Acesse https://github.com/SEU_USUARIO/fraga-dashboard
   Ou no terminal: git log --oneline -5

P: "Como usar SSH em vez de HTTPS?"
R: Mais seguro! Gere chave:
   ssh-keygen -t ed25519 -C "seu_email@example.com"
   Copie a chave em ~/.ssh/id_ed25519.pub para GitHub → Settings → SSH keys
   Depois use: git@github.com:USUARIO/fraga-dashboard.git

P: "Preciso fazer deploy automático?"
R: Configure os secrets em GitHub → Settings → Secrets:
   • DEPLOY_KEY (sua chave SSH privada)
   • DEPLOY_HOST (IP do servidor)
   • DEPLOY_USER (usuário SSH)
   O workflow já está preparado!
